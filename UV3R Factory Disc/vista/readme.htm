USB cable User Installation Manual

For Windows vista

( 11/2009)

1.Power on your computer and boot to Windows. 2.Locate the USB port of your computer and plug in the USB cable.Windows will detect the new device and will initiate the Found New Hardware Wizard prompting you to install the software for the detected USB Device. 3.Click Cancel to continue.

4.right clik computer→select Manage

5.Computer Management→Device Manager→right clik USB-Serial controller→Update Driver Software

6.select Browse my computer for dirver software

7.select Let me pick from a list of device drivers on my computer

8.click Next to continue

9.click Have Disk…

10.click Browse

11.click vista

12 select “serspl”

13.click ok to continue

14. click Next to continue

15. installing…

16.select Install this driver software anyway

17. install successfully

18. Verifying the Installation

You should see the Prolific USB-to-Serial Comm Port (COM *) device listing. This also means that Windows has assigned the device to COM * port.
